## 📖 Project Description

RecipeMaster AI Server is a RESTful API built to power the RecipeMaster AI platform. It handles authentication, recipe management, AI chatbot communication, image uploads, reviews, and secure database operations while serving data to the frontend application.

## ⚙️ Installation Guide

### Clone the repository

```bash
git clone https://github.com/apurbochaki-lab/RecipeMaster-AI-Server.git
```

### Move into project directory

```bash
cd RecipeMaster-AI-Server
```

### Install dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file:

```env
PORT="server_port"
MONGODB_URI="mongodb_uri_link"
NEXT_PUBLIC_CLIENT_URL="client_side_url"

```

### Run the server

```bash
npm run dev
```
